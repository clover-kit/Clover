# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-12

### Added
- **New Presets**: Added 7 handcrafted text animation presets:
    - *Modern Fade*: Clean, blurred entry.
    - *Terminal*: Classic typewriter effect with cursor.
    - *Editorial*: Elegant, staggered rise.
    - *System Glitch*: Digital artifacting and displacement.
    - *Pop In*: Spring-loaded scale animation.
    - *Cinematic*: Slow, misty reveal.
    - *Elastic*: Big movement with a bouncy finish.
- **Interactive Playground**:
    - Live preview of all animations.
    - "Copy Code" button for instance generation.
    - URL sharing support (presets and settings persist in URL).
    - Keyboard navigation (`Cmd+K` / `Ctrl+K`) to cycle presets.
- **Documentation**: Added `/docs` route with "How to Guide" and installation instructions.
- **CLI Support**: Added support for installing presets via `npx shadcn@latest add ...`.
- **UI/UX**:
    - Complete visual overhaul with "It's Hover" aesthetic.
    - Dark mode support (System/Light/Dark toggle).
    - Dynamic GitHub star count in header.
    - Responsive design for mobile and desktop.
